# Novidades na Versão 1.0.0

Esta é a primeira versão oficial do GENESI-MVC-TEMPLATE, focada em fornecer uma base sólida e performática para o desenvolvimento de aplicações web em PHP puro, seguindo o padrão MVC.

## Principais Recursos e Melhorias:

- **Estrutura MVC Completa**: Implementação clara e funcional do padrão Model-View-Controller para uma separação de responsabilidades eficiente.
- **Autoloading PSR-4 com Composer**: Gerenciamento de dependências e carregamento automático de classes para um código organizado e moderno.
- **Roteamento por Convenção**: Sistema de roteamento simples e intuitivo que mapeia URLs diretamente para Controllers, agilizando o desenvolvimento.
- **Tratamento de Erros 404**: Redirecionamento automático para uma página de erro 404 quando um Controller não é encontrado.
- **Boas Práticas de Código**: Adoção de convenções de nomenclatura e estrutura de diretórios para manter a consistência e legibilidade do código.
- **Configuração Inicial para MySQL**: Base pronta para integração com bancos de dados MySQL.
- **Documentação Detalhada**: README abrangente explicando a arquitetura, como usar e as tecnologias envolvidas.

## Como Atualizar:

Esta é a versão inicial, portanto, não há um processo de atualização a partir de versões anteriores. Para começar, siga as instruções de "Como Utilizar este Template" no README principal.

## Próximos Passos:

Estamos sempre buscando melhorar e expandir as funcionalidades do GENESI-MVC-TEMPLATE. As próximas versões podem incluir:

- Suporte a ORM (Object-Relational Mapping) para interação com banco de dados.
- Sistema de autenticação e autorização.
- Ferramentas de validação de formulários.
- Melhorias no tratamento de erros e logging.
